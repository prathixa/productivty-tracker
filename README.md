## Project Overview:
Developed a comprehensive productivity monitoring system that tracks user web activity via a browser extension and visualizes data through a high-end MERN stack dashboard. The project bridges the gap between raw data collection and actionable user insights.
Technical Stack:

Frontend: React.js, Chart.js, Axios

Backend: Node.js, Express.js

Database: MongoDB
